# 首页门户区（HomePortal）实现方案

## 目标

复刻参考博客首页文章列表上方的门户区，包含：
1. 头像涟漪（上下班状态感知 + MagicRings 动效）
2. 昵称 + 职业标签
3. 打字机 bio
4. 刮刮卡式站点统计（悬停刮开揭示详细数据）

## 可复用资产

| 文件 | 状态 | 说明 |
|------|------|------|
| `TypewriterText.astro` | ✅ 已有 | 直接复用 |
| `MagicRings.ts` | ✅ 已有 | 直接复用 |
| `types/config.ts` | ✅ 已有 | `displayName`、`occupation`、`avatarOffWork`、`workHours` 类型已定义 |
| `SiteStats.astro` | ⚠️ 侧边栏版 | 需新建刮刮卡版 |

## 实现步骤

### Task 1: 更新配置文件

**文件：** `src/config/profileConfig.ts`

新增字段：
```ts
displayName: "团子",           // 首页展示名（留空则用 name）
occupation: "[技术博主 / 生活记录者]",  // 职业标签
avatarOffWork: "",             // 下班头像 URL（用户稍后提供，保留空字符串占位）
bio: [                         // 改为数组，支持多条循环打字
  "海棠花未眠，老陈总在我身边",
  "记录更多，言语更少",
],
```

**文件：** `src/config/siteConfig.ts`

新增字段：
```ts
workHours: {
  start: 9,
  end: 18,
  workDays: [1, 2, 3, 4, 5],
},
```

### Task 2: 新建刮刮卡站点统计组件

**新建文件：** `src/components/widget/PortalStats.astro`

功能：
- 默认层：显示访客数 + 访问量（从 Waline API 获取，你的博客已有 Waline）
- 揭示层（鼠标悬停从左到右刮开）：文章数 · 分类数 · 标签数 · 总字数
- 粒子背景动画（桌面端）
- 竖线光标跟随鼠标

数据源：
- 访客/访问量：通过 Waline API 获取（`commentConfig.waline.serverURL` + `/api/comment/type?type=1&path=/` 等端点）
- 文章/分类/标签/字数：构建时计算（复用 `content-utils.ts`）

**用户确认：** 使用 Waline 统计，头像提供两张（上班/下班），职业标签为 "[技术博主 / 生活记录者]"

### Task 3: 新建门户区组件

**新建文件：** `src/components/layout/HomePortal.astro`

结构：
```
┌─────────────────────────────┐
│      [头像 + 涟漪动效]        │
│      [状态按钮: 上班中/下班]    │
│                             │
│         团子                 │
│    [技术博主 / 生活记录者]     │
│    [打字机 bio 文字]          │
│                             │
│     ── 站点统计 ──           │
│   [刮刮卡统计组件]            │
│                             │
│     ── 博客文章 ──           │
└─────────────────────────────┘
```

依赖：
- `MagicRings.ts` - 涟漪动效
- `TypewriterText.astro` - 打字机
- `PortalStats.astro` - 刮刮卡统计
- `ImageWrapper.astro` - 头像图片
- `profileConfig` / `siteConfig` - 配置数据

### Task 4: 集成到首页

**修改文件：** `src/pages/[...page].astro`

在 `<PostPage>` 前插入 `<HomePortal>`（仅第一页显示）：
```astro
<MainGridLayout>
  {page.currentPage === 1 && <HomePortal />}
  <PostPage page={page} />
  ...
</MainGridLayout>
```

### Task 5: 新增 i18n 键

**修改文件：** `src/i18n/i18nKey.ts` + 所有语言文件

新增键：
- `siteStatsVisitorPrefix` - "您是造访本站的第 "
- `siteStatsVisitorSuffix` - " 位旅人，很高兴与你相遇。"
- `siteStatsPageViewPrefix` - "小破站访问量为 "
- `siteStatsPageViewSuffix` - ""
- `siteStatsLoading` - "--"
- `siteStatsHoverHint` - "悬停查看更多"

## 验证方式

1. `pnpm dev` 启动开发服务器
2. 访问首页，确认门户区显示在文章列表上方
3. 检查头像涟漪动效（上班时间应显示粉色涟漪 + "上班中"状态）
4. 检查打字机效果循环播放
5. 鼠标悬停刮刮卡，确认从左到右揭示统计数据
6. 切换暗色模式，确认样式正常
7. 移动端响应式检查
8. `pnpm build` 构建成功
