这个文件是“生活（bangumi）”内容集合的示例条目，你可以照着复制一份来新增自己的记录；也可以在确认学会后直接删除本文件。

## 字段说明（Frontmatter）

### title（必填）
显示在列表里的主标题。

示例：
- `title: "示例：如何填写生活条目（可删除）"`

### name_cn（可选）
中文别名/第二标题。用于你想同时保留原名 + 中文名的场景（比如日文/英文作品名）。

示例：
- `name_cn: "生活条目填写示例"`

### category（必填，枚举）
条目类型，用于在“生活”页面做分组/筛选。

可选值：
- `anime`：影视
- `book`：书籍
- `music`：音乐
- `game`：游戏
- `real`：三次元（真人/综艺/纪录片等）

示例：
- `category: "anime"`

### status（必填，1-5）
条目状态，用于筛选“想看/看过/在看/搁置/抛弃”等。

含义：
- `1`：想看 / 想读 / 想听 / 想玩
- `2`：看过 / 读过 / 听过 / 玩过
- `3`：在看 / 在读 / 在听 / 在玩
- `4`：搁置
- `5`：抛弃

示例：
- `status: 2`

### image（必填）
封面图。可以是：
- 远程图片 URL（推荐，最快）
- 本地图片（放到 `public/` 下，然后用 `/xxx.webp` 这种路径）

示例：
- `image: "https://re.tsh520.cn/zl/zbd.webp"`

### link（必填）
点击条目时跳转的链接。可以是：
- 站内文章：`/posts/xxx/`
- 站内页面：`/bangumi/`
- 外部链接：`https://...`

示例：
- `link: "/posts/some-article/"`
- `link: "https://bgm.tv/subject/xxxx"`

### score（可选，0-10）
评分（支持小数）。

示例：
- `score: 8.5`

### comment（可选）
一句话短评/备注。

示例：
- `comment: "画面很棒，配乐也很戳我。"`

### tags（可选）
标签数组，用于你自己做标记或未来扩展筛选。

示例：
```yaml
tags:
  - 治愈
  - 必看
```

### published（可选）
日期（用于“最近更新”/排序等场景）。不填时会被当作非常早的时间，可能不会出现在“最近更新”里。

示例：
- `published: 2026-04-03`
