# ⚠️ 重要提醒（基于文档）

1. 代码一定要手敲：视频可以倍速看，但代码必须一行一行手敲，不要复制粘贴
2. 不要追求完美：三个月时间有限，优先掌握🌕必须学内容，其他入职后再补
3. 尽早投递：不要等全部学完再投，第11周就开始投递，用面试发现问题
4. 项目比八股文重要：面试官更看重你做过什么项目，而不是背了多少题
5. 善用AI工具：遇到不懂的概念或Bug，用AI辅助理解（但别完全依赖）
6. 文档明确说："学完阶段1、3、5就可以开始找工作了"，你正在正确的路上
