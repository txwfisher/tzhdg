## 后台友链管理

- 新增后台友链管理页面（`/admin/friends/`），支持通过粘贴文本自动解析并添加友链
- 支持格式：`站点名称: xxx / 站点描述: xxx / 站点链接: xxx / 头像链接: xxx`
- 实时预览功能，输入时自动显示友链卡片预览
- 友链数据存储在独立的 GitHub Gist 中（`friends.json`）
- 友链页面自动从 Gist 加载外部友链，与本地友链合并显示
- 后台首页添加友链管理入口

## 主题切换修复

- 修复从其他页面进入说说页面后，导航栏亮暗色切换按钮点击无反应的问题
- 原因：FloatingDock 使用模块脚本，Swup 替换内容后事件监听器丢失
- 将主题切换改为 document 级别事件委托，不受 Swup DOM 替换影响

## 置顶说说页面修复

- 修复置顶页面图片无法点击放大的问题：添加 `data-fancybox` 和 `data-src` 属性
- 修复暗色模式下说说文字不可见的问题：`var(--text-color)` 改为 `var(--deep-text)`
- 修复暗色模式下"置顶说说"标题不可见的问题

## 首页导航栏遮挡修复

- 修复首页 "Dumpling" 文字 VFX 特效遮挡导航栏的问题
- 将 VFX canvas 挂载目标从 `document.body` 改为 hero 元素内部

## 影视游戏页面按钮样式改版

- Tab 切换和筛选按钮改为与更新日志一致的胶囊风格
- 添加滑动指示器动画效果
- 修复切换 Tab 时筛选指示器未初始化的问题

## 友链页面暗色模式修复

- 修复暗色模式下标签筛选按钮选中后背景为白色、文字也为白色导致看不清的问题
- 按钮激活态添加 `dark:text-neutral-900` 确保暗色模式下文字为深色
