## 笔记后端系统（GitHub Gist）

- 新增 `externalNotebooksConfig` 配置，每个笔记本对应独立 Gist 仓库
- 新增 `/admin/notebooks/` 笔记本管理页面（选笔记本 → 写 Markdown → 存 Gist）
- 新增 `/admin/notebooks/detail/` 笔记编辑页面（Markdown 编辑器 + 实时预览）
- 新增 `/life/notebooks/remote-entry/` 远程笔记阅读页（样式与本地笔记一致）
- 前端笔记本页面自动合并显示远程笔记（通过 Gist Raw URL 获取，无需 API 认证）
- 笔记编辑器支持工具栏快捷插入（加粗、斜体、代码块、链接、图片、引用、列表）
- 6 个快速笔记模板（每日总结、日记、读书笔记、灵感、待办、空白），可在配置文件中自定义

## 主题切换改造

- 导航栏骷髅头主题切换改为太阳/月亮图标按钮（旋转 + 透明度过渡动画）
- 首页封面图开关从笑脸拨动开关改为骷髅头滑块样式
- 骷髅头滑块支持二态切换（封面开启/关闭）

## 首页样式调整

- 封面关闭时文章卡片显示 2.5px 黑线框（暗色模式为白线框）
- 分类/归档按钮边框同步调整为 2.5px + var(--deep-text) 颜色
- 骷髅头按钮左边缘与文章卡片黑线对齐
- 导航栏按钮 hover 时不再显示背景色

## 欢迎卡片改造

- 翻转逻辑从拖拽点击改为「查看详情」按钮翻转
- 背面新增「返回」按钮翻回正面
- 按钮下方显示「↕ 拖拽弹射」提示文字
- 移除卡片显示时的背景暗化效果（遮罩层改为透明）
- 正面摘要和背面列表自动合并远程说说和笔记数据

## 最近更新小组件

- 侧边栏最近更新小组件自动获取远程说说和笔记数据
- 使用 Gist Raw URL 获取数据（Secret Gist 也能访问）
- 整体条数限制为 10 条（本地 + 远程合并）
- 远程数据最多显示 3 条，其余留给本地数据
- 支持桌面端和移动端两个实例同时更新
