## 文章系统 (posts)

### 路径
`src/content/posts/`

### 创建文章

在 `src/content/posts/` 下创建 `.md` 或 `.mdx` 文件：

```markdown
---
title: 我的第一篇文章
published: 2026-04-18
description: 这是文章描述
tags:
  - 标签1
  - 标签2
category: 分类名称
pinned: false
image: https://example.com/cover.jpg
---

文章正文内容...
```

### 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| `title` | 是 | 文章标题 |
| `published` | 是 | 发布日期 (YYYY-MM-DD) |
| `description` | 否 | 文章描述，显示在列表页 |
| `tags` | 否 | 标签数组 |
| `category` | 否 | 分类名称 |
| `pinned` | 否 | 是否置顶 |
| `image` | 否 | 封面图 |
| `draft` | 否 | 是否为草稿 (草稿不显示) |
| `comment` | 否 | 是否开启评论 (默认 true) |
| `updated` | 否 | 更新日期 |

### 子目录

支持在 `posts/` 下创建子目录组织文章：

```
posts/
├── 技术/
│   └── docker.md
├── 生活/
│   └── daily.md
└── 示例.md
```

## 日常动态 (moments)

### 路径
`src/content/moments/`

### 创建动态

在 `src/content/moments/` 下创建 `.md` 文件：

```markdown
---
author: 团子和蛋糕
avatar: https://example.com/avatar.jpg
published: 2026-04-18
images: photo1.jpg, photo2.jpg, photo3.jpg
tags:
  - 日常
location: 上海
device: iPhone 15 Pro
---

动态正文内容...
```

> 注意：`images` 字段使用**逗号分隔**的字符串格式，不使用数组。

### 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| `author` | 否 | 作者名 |
| `avatar` | 否 | 头像 URL |
| `published` | 是 | 发布时间 |
| `images` | 否 | 图片数组 |
| `tags` | 否 | 标签 |
| `location` | 否 | 地点 |
| `device` | 否 | 设备 |

## 生活记录 (life)

生活记录是本博客的核心功能，包括健康追踪、想法管理、打卡、规划等多个模块。

### 健康记录

#### 运动 (workout)
**路径**: `src/content/life/health/workout/`

```markdown
---
date: 2026-04-18
workoutType: 户外跑步
runKm: 5.2
workoutMinutes: 30
stepsToday: 8000
---
```

#### 睡眠 (sleep)
**路径**: `src/content/life/health/sleep/`

```markdown
---
date: 2026-04-18
sleepHours: 7.5
sleepQuality: 4
---
```

| 字段 | 说明 |
|------|------|
| `sleepHours` | 睡眠时长 (小时) |
| `sleepQuality` | 睡眠质量 (1-5) |

#### 饮食 (food)
**路径**: `src/content/life/health/food/`

```markdown
---
date: 2026-04-18
allowedFoods:
  - 蔬菜
  - 水果
avoidFoods:
  - 辛辣食物
  - 甜食
---
```

### 笔记本

笔记本功能用于创建个人日记本，每个笔记本是一个文件夹。

#### 创建步骤

1. 在 `src/content/life/notebooks/` 下创建文件夹
2. 在文件夹内创建 `_index.md` 或 `_index.json` 文件
3. 在文件夹内创建 `.md` 文件作为日记条目

#### 示例

```
notebooks/
└── 日记/
    ├── _index.md
    ├── 2026-04-18.md
    └── 2026-04-17.md
```

#### _index.md 内容

```markdown
---
name: 日记
cover: ""
summary: 记录每天的生活点滴
---
```

#### 日记条目内容

```markdown
---
date: 2026-04-18
---

今天发生了一些有趣的事情...
```

## 番组计划 (bangumi)

**路径**: `src/content/bangumi/`

支持 `.md`, `.yaml`, `.yml` 格式：

```markdown
---
title: 攻壳机动队
name_cn: 攻壳机动队 Stand Alone Complex
category: anime
status: 2
image: https://example.com/cover.jpg
link: https://bgm.tv/subject/1
score: 10
tags:
  - 科幻
  - 神作
published: 2026-01-01
---
```

### status 状态码

| 值 | 状态 |
|----|------|
| 1 | 想看 |
| 2 | 看过 |
| 3 | 在看 |
| 4 | 搁置 |
| 5 | 抛弃 |

### category 类型

| 值 | 类型 |
|----|------|
| anime | 动画 |
| book | 书籍 |
| music | 音乐 |
| game | 游戏 |
| real | 影视 |

## 侧边栏组件

侧边栏组件在 `src/config/sidebarConfig.ts` 中配置。

### 可用组件

| 组件 | 说明 |
|------|------|
| `profile` | 用户资料 |
| `announcement` | 公告 |
| `music` | 音乐播放器 |
| `categories` | 分类 |
| `tags` | 标签 |
| `lifeStats` | 生活统计 |
| `relationship` | 恋爱计时 |
| `quoteOfTheDay` | 今日一言 |
| `recentItems` | 最近更新 |
| `stats` | 站点统计 |
| `calendar` | 日历 |
| `sidebarToc` | 文章目录 |
| `siteHeatmap` | 更新热力图 |
| `advertisement` | 广告位 |

### 组件配置示例

```typescript
{
  type: "lifeStats",
  enable: true,
  position: "sticky",
  showOnPostPage: true,
  showOnNonPostPage: true,
}
```

## 评论系统

博客使用 Waline 作为评论系统。

评论功能在以下位置启用：
- 文章详情页 (默认开启，可通过 frontmatter 关闭)
- 日常动态页 (`/moments/`)
- 留言板 (`/guestbook/`)

## 配置文件

### siteConfig.ts

主要配置文件，位于 `src/config/siteConfig.ts`

| 配置项 | 说明 |
|--------|------|
| `title` | 站点标题 |
| `description` | 站点描述 |
| `themeColor` | 主题色配置 |
| `navbar` | 导航栏配置 |
| `favicon` | 网站图标 |
| `pages` | 页面开关 (sponsor/guestbook/bangumi) |
| `postListLayout` | 文章列表布局 |
| `pagination` | 分页配置 |
| `mapConfig` | 地图配置 (高德地图) |

### 导航栏配置

在 `src/config/navBarConfig.ts` 中配置顶部导航菜单。

## 常见问题

### Q: 如何禁用某个页面？

在 `siteConfig.ts` 中修改 `pages` 配置：

```typescript
pages: {
  sponsor: false,    // 禁用赞助页
  guestbook: false,  // 禁用留言板
  bangumi: false,    // 禁用番组页
}
```

### Q: 如何添加自定义页面？

在 `src/pages/` 下创建新的 `.astro` 文件即可。

### Q: 如何修改主题色？

在 `siteConfig.ts` 中修改 `themeColor.hue` 值 (0-360)。
