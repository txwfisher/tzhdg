# 数据库基础概念

## 定义

- **数据库（DataBase，简称DB）**：存储和管理数据的仓库。
- **数据库管理系统（DataBase Management System，简称DBMS）**：操作和管理数据库的大型软件。
- **SQL（Structured Query Language）**：结构化查询语言，操作关系型数据库的编程语言，定义了操作所有关系型数据库的统一标准。

三者关系：程序员给 DBMS 发送 SQL 语句，再由 DBMS 操作数据库中的数据。

## 主流数据库排名

| 数据库 | 特点 |
|---|---|
| Oracle | 大型收费数据库，Oracle 公司产品 |
| MySQL | 开源免费的中小型数据库，互联网公司使用最广泛 |
| SQL Server | Microsoft 收费中型数据库，C#/.NET 常用 |
| PostgreSQL | 开源免费的中小型数据库 |
| DB2 | IBM 大型收费数据库 |
| SQLite | 嵌入式微型数据库（Android 内置） |
| MariaDB | MySQL 的分支，开源免费，与 MySQL 兼容性好 |

## 注意事项

- MySQL 目前由 Oracle 维护，分为社区版（免费）和企业版（收费）。
- 所有关系型数据库都遵循统一的 SQL 标准，学会 SQL 即可操作 MySQL、Oracle、SQL Server 等。
- 课程大纲：MySQL 概述 → SQL（DDL/DML/DQL）→ 多表设计 → 多表查询 → 事务 → 索引。
