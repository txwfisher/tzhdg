|     快捷键      |     功能      |
| :----------: | :---------: |
|    Ctrl+s    |   保存当前文件    |
|      F2      |    编辑文件名    |
|    Ctrl+K    |    插入链接     |
|      F1      |    查看帮助     |
|    Ctrl+W    |   关闭当前标签页   |
|    Ctrl+M    |    插入模版     |
| Ctrl+Shift+F |    表格居中     |
|    Ctrl+N    | 在对应文件夹创建新文件 |
|              |             |
|              |             |
